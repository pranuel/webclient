## Install required tools

```shell
npm install -g yarn
```

## Install dependencies

```shell
yarn [install]
```

## Build

```shell
yarn build
```

_(defaults to development build)_

### Development

```shell
yarn build:dev
```

### Production

```shell
yarn build:prod
```

## Run

```shell
yarn start
```

## Check for outdated yarn packages

```shell
yarn outdated
```

## Upgrade interactively

```shell
yarn upgrade-interactive [--latest]
```
